Lognormal Exposure Sheet for Excel

Enter exposure readings, generate Excel formulas, paste the whole table into Excel, and produce a chart-ready fitted lognormal distribution curve. This no-ranking version fits directly from the mean and standard deviation of logged readings.

1. Spreadsheet input and formulas

Only positive exposure readings can be log-transformed. Blank, zero, and negative values are ignored in the fitted curve.

2. Browser preview of fitted curve

This graph is only a browser preview. The Excel graph is produced from the pasted columns labelled Chart X and Chart Y: lognormal PDF.

3. Excel graph

  1. Click Copy Excel-ready sheet.
  2. Paste into Excel cell A1.
  3. Select columns E:F, including Chart X and Chart Y: lognormal PDF.
  4. Choose Insert → Scatter → Scatter with Smooth Lines.
