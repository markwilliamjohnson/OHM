Exposure Readings → Excel Lognormal Distribution Sheet

Enter occupational exposure readings, generate a spreadsheet-like table containing Excel formulas, copy the whole range, paste into Excel, and use the chart-ready output columns to create a lognormal probability distribution graph.

Paste the copied table into cell A1 in Excel. The formulas use Excel functions such as LN, EXP, AVERAGE, STDEV.S, NORM.DIST, LOGNORM.DIST, and COUNT. The final two columns are designed as the X and Y series for an Excel scatter chart.

2. Preview graph in browser

This browser graph is only a preview. The Excel graph is produced from the pasted columns labelled Chart X and Chart Y: lognormal PDF.

3. Excel chart instructions

After pasting into Excel:

  1. Select the populated cells in columns I:J, starting with the headers Chart X and Chart Y: lognormal PDF.
  2. Choose Insert → Scatter → Scatter with Smooth Lines.
  3. Set the chart title to Lognormal exposure distribution.
  4. The chart uses the fitted geometric mean and geometric standard deviation from the readings.

Optional: select columns A:B and create a second scatter plot of readings against their natural logs to show the transformation.
